# Metaoneer KGF 2차 프로젝트

## Klaytn NFT 통합 마케팅 서비스

- NFT를 등록 및 수정, 조회할 수 있으며, NFT를 기반으로 Event 생성이 가능한 마케팅 서비스입니다.

- 대시보드를 통해 다가오는 이벤트를 확인하고, 종료된 이벤트를 확인하여 보상을 지급할 수 있습니다.

---

```
npm install --legacy-peer-deps
```
